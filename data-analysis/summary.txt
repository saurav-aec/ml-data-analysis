Exploratory Data Analysis (EDA) on Malnutrition Dataset 
-------------------------------------------------------

Data Set - https://www.kaggle.com/ruchi798/malnutrition-across-the-globe

Description - 
------------
Malnutrition data has been analyzed - 
> Data spread across Income Groups
> Rate of growth comparison between malnutrition types
> Coorelation Analysis across the malnutrition types
> Data Distribution of malnutrition types

Technology used -
---------------
R Markdown
Plotly
